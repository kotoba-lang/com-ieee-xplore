# Ieee_xplore Clean Room Actor

Clean-room API-compatible implementation of the ieee_xplore physical substrate/core protocol, backed by Datomic and Py Kotodama WASM.

## Provenance

Relocated 2026-07-04 from `etzhayyim/root/20-actors/ieee_xplore-compat` to
`kotoba-lang/com-ieee-xplore` per the org-taxonomy library-placement rule (any
library/substrate code belongs in `kotoba-lang`, ADR-2606302300), following
the same relocation pattern as `kami-nv-compat` (ADR-2607020130). See
ADR-2607041500 for the full ~1,027-repo migration plan and naming convention.
